# 向僵尸开炮游戏机器人

一个用于自动化游戏操作的Python脚本，支持自动刷环球、刷卡关等功能。

## 功能特性

### 核心功能
- 🎯 **自动刷环球** - 自动加入队伍并战斗
- 🎮 **自动主线** - 自动挑战主线关卡
- ⚔️ **普通远征** - 自动刷普通远征模式
- 🔥 **超级远征** - 自动刷超级远征模式
- ⚡ **技能优先级** - 支持设置5个优先技能，按顺序匹配
- � **消费模式** - 支持选择「我是土豪」或「我是穷B」模式，影响远征策略
- � **图像识别** - 使用OpenCV进行模板匹配
- 🖥️ **图形界面** - 友好的Tkinter GUI操作界面
- 💾 **配置保存** - 自动保存技能配置，下次启动自动加载
- ⌨️ **快捷键** - 按ESC键暂停脚本
- 📏 **窗口大小调整** - 一键调整游戏窗口为推荐的542*1010大小

### 技术特点
- 支持Windows PC端微信小程序
- 支持多模板匹配（每个技能可设置多个模板）
- 支持PyInstaller打包为可执行文件
- 智能窗口查找和激活
- 异常处理和错误提示
- 实时状态显示

- Python 3.7+

```bash

```

```
├── game_bot.py        # 主脚本文件
```

```bash

```

#### 方法2：命令行参数
```bash
python game_bot.py --game_title "向僵尸开炮" --mode 0 --battle_count 10
```

## 使用指南

### GUI界面说明
1. **游戏窗口标题** - 输入游戏窗口的标题（默认为"向僵尸开炮"）
2. **模式选择** - 选择运行模式：
   - **环球** - 自动刷环球（进别人队伍）
   - **主线** - 自动挑战主线关卡
   - **普通远征** - 自动刷普通远征
   - **超级远征** - 自动刷超级远征
3. **消费模式** - 选择消费策略（仅在环球、普通远征、超级远征模式下显示）：
   - **我是土豪** - 不检查门票，直接挑战
   - **我是穷B** - 检查门票，避免不必要的消耗
4. **战斗次数** - 设置战斗次数（0表示无限循环）
5. **战斗时间** - 设置每次战斗的最大时间（0表示无限制）
6. **优先技能** - 从下拉框选择5个优先技能（按优先级从上到下）
7. **开始/停止** - 控制脚本运行
8. **调整窗口大小** - 一键将游戏窗口调整为推荐的542*1010大小，提高识别准确率

### 模板图片准备
在`templates`目录中添加以下模板图片：
- `skill.png` - 技能按钮
- `battle.png` - 战斗按钮
- `battling.png` - 战斗中状态
- `return.png` - 返回按钮
- 其他界面元素模板

**注意**：为了提高识别准确率，建议使用游戏窗口的原始分辨率，窗口大小推荐使用542*1010。

- 车

## 命令行参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--game_title` | 游戏窗口标题 | "向僵尸开炮" |
| `--mode` | 运行模式 (0: 环球, 1: 主线, 2: 普通远征, 3: 超级远征) | 0 |
| `--battle_count` | 战斗次数 (0表示无限) | 0 |
| `--battle_time` | 战斗时间限制(秒) | 0 |

## 注意事项

1. **兼容性** - 使用OpenCV图像对比，可能对不同分辨率和设备有兼容性问题
2. **干扰** - 游戏过程中尽量不要操作鼠标和键盘，以免干扰脚本运行
3. **权限** - 确保脚本有足够的权限控制鼠标和键盘
4. **安全** - 本脚本仅供学习参考，请勿用于商业用途或破坏游戏平衡
5. **更新** - 游戏更新后可能需要更新模板图片

## 常见问题

### Q: 脚本找不到游戏窗口
A: 确保游戏窗口标题正确，且窗口未被最小化

### Q: 技能识别失败
A: 检查`templates`目录中的技能模板图片是否正确

### Q: 脚本运行不稳定
A: 调整游戏窗口分辨率为推荐的542*1010，确保模板图片清晰
